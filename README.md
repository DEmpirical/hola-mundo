# Hola Mundo

Una sencilla aplicación web que muestra un mensaje de "Hola Mundo" con diseño moderno y un contador interactivo.

## 🚀 Características

- Diseño responsivo con gradiente atractivo
- Interactividad: contador de clics que persiste con localStorage
- Código limpio: HTML, CSS y JavaScript en un solo archivo
- Sin dependencias externas

## 📦 Estructura

```
hola-mundo/
├── index.html    # Aplicación completa (HTML + CSS + JS)
└── README.md     # Este archivo
```

## 🛠️ Cómo usar

1. Abre `index.html` en cualquier navegador moderno
2. ¡Haz clic en el botón para aumentar el contador!
3. El contador se guarda automáticamente en el navegador

## 🌐 Despliegue

Puedes alojar este archivo estático en:

- GitHub Pages
- Netlify
- Vercel
- Cualquier servidor HTTP estático

Simplemente sube `index.html` y accede a él.
